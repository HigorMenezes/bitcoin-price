# bitcoin-price

Enviar o preço do Bitcoin para o e-mail

### .env template

```.env
DB_HOST=
DB_USERNAME=
DB_PASSWORD=
DB_DATABASE=
DB_DIALECT=
DB_STORAGE=

APP_PORT=3030
APP_SECRET=
```

### Tabela de status

- Status de 100 a 199 são status de OK;
- Status de 200 a 299, são status de pendência;
- Status de 300 a 399, são status de problemas.
